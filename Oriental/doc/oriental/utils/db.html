uri4sqlite(p: pathlib.Path)¶
SQLite supports file URIs only with authority being empty or ‘localhost’,

except if compiled with SQLITE_ALLOW_URI_AUTHORITY - which is not the case by default, and not the case for Python’s sqlite.dll However, one can trick SQLite to accept URIs of network files by passing URIs that start with 4 slashes!
