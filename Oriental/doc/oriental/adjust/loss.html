API¶

loss functors for oriental.adjust

Note that adjust considers the first and second derivatives of loss functions. Thus, to ease convergence, a good loss function should not only be smooth, but also have smooth first and second derivatives.

class IUser((object)arg1) → None¶

Bases: oriental.adjust.loss._loss.Base

Loss and its derivatives computed by user

Usage:

class MyLoss( IUser ):
    def __init__( self ):
        super().__init__()
    def Evaluate( self, sq_norm, out ):
        # Trivial loss would be:
        out[0] = sq_norm
        out[1] = 1.
        out[2] = 0.
Evaluate((IUser)arg1, (object)sq_norm, (numpy.ndarray)out) → None :¶

sq_norm is the 2-norm of the residual vector. Set the 3 elements of out based on sq_norm and my loss function. The i-th element of out shall contain the i-th derivative of the user-defined loss-function, evaluated at sq_norm
