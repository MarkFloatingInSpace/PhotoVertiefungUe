deactivate¶

deactivate observations

class DeactivateTiePts(block: oriental.adjust._adjust.Problem, solveOpts: oriental.adjust._adjust.Options, cameras: dict, images: dict, tieObjPts: dict, tieImgObsDataType: type = <class 'NoneType'>, minResBlocksPerPho: int = 20, minResBlocksPerPoint: int = 2)¶

Remove tie image point observations, tie object points, images, and cameras from the block. Removes no other observation types. Removes tie object points, images, and cameras only if observed exclusively by tie image points.
