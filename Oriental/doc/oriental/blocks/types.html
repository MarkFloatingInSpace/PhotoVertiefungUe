createCompact(Base: type, attribs: str | seq(str)) → type¶
returns a class derived from Base with attribs as additional attribute(s)

If Base has no __dict__, then the returned class will not have one, either. The returned class is pickleable. Base may be the result of a preceding call to createCompact, thereby supporting inheritance.
