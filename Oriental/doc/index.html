OrientAL manual¶

OrientAL aims to provide the necessary tools for manual, semi- and fully automatic, photogrammetric image orientation, camera calibration, and object reconstruction. It provides functionality

  • as high-level command-line Scripts,

  • as lower level functions of the Python-package oriental, and

  • by the GUI-program MonoScope.
