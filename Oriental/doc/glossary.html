Glossary¶

$ORIENTAL

the directory into which your distribution of OrientAL has been extracted. See Directory Structure.

cameras.sqlite

SQLite data base with information on the sensors of various camera models.

cost function
cost functions

another function

CSV

Comma-separated values, a text file format (usually with .csv extension)

DLL load failed: The specified module could not be found

See section Prerequisites for Installations on Windows

DSM

Digital surface model

DTM

Digital terrain model

exterior orientation

see Spatial Similarity Transformation

failed to create process.

see Installation of Additional Python Packages

GDAL

The Geospatial Data Abstraction Library is used throughout OrientAL for raster data access.

I’m sorry, Dave. I’m afraid I can’t do that.

The prologue of OrientAL license errors.. See License file.

interior orientation

see Spatial Similarity Transformation

lba.sqlite

SQLite data base containing the meta data of the aerial image archive of UHA

loss function
loss functions

a function that returns a value

Open Geospatial Consortium
OGC

The OGC is an international industry consortium of 478 companies, government agencies and universities participating in a consensus process to develop publicly available interface standards.

ORIENT

ORIENT is a universal program system mainly intended to do rigorous adjustments in the field of photogrammetric point determination.

OrientAL

German for: Orientierung archäologischer Luftbilder.

OrientAL shell

See section OrientAL shell.

PyScripter

The Python IDE that comes with OrientAL.

SpatiaLite

Extension module for SQLite that adds Spatial SQL capabilites mostly conforming to the OGC-standard Simple Feature Access - Part 2: SQL Option. SpatiaLite roughly is to SQLite what PostGIS is to PostgreSQL.

SQLite

SQLite is a software library that implements a self-contained, serverless, zero-configuration, transactional SQL database engine.

UHA

Institute of Prehistoric and Historical Archaeology, University of Vienna
