Scripts provide the high-level functionality of OrientAL, aiming at utmost levels of automation. All scripts can be found in the directory $ORIENTAL/Scripts/. For every Python script file (e.g. relOri.py) in that directory, an executable file of the same name exists (e.g. relOri.exe). While the Python script files are supposed to be passed as the first argument to $ORIENTAL/python/python.exe, the corresponding executable files can be called directly with their according arguments. For simple invocation of OrientAL scripts, call them from within an OrientAL shell.
